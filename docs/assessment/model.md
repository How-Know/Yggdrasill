# 구조 요약 및 정의

## 0. 한눈에 보는 구조
- **비개입 변수(통제 어려운 것)**: 마음, 재능, 운, 학습 환경
  - 마음 하위 기준 구조: 기질, 성향, 신념
- **개입 가능 변수(통제 가능한 것)**
  - 통합적 수학 역량: 능력 지표(질문 구성 능력, 사고력, 논리력, 문제해결력, 메타인지)
  - 정신
    - 능력 지표: 정서 조절 능력, 동기 조절 능력, 인지적 회복력,
      과제 가치 인식(준안정 경향), 내재적 동기(준안정 경향), 지속 의지(준안정 경향)
    - 상태 지표: 불안 수준, 흥미, 즐거움, 좌절, 자신감, 긴장, 피로
  - 행동
    - 능력 지표: 전략 실행 능력, 오류 회복 능력, 지속성(문제 지속성)
    - 상태 지표: 시도 빈도, 회피 반응, 질문 빈도
- **성과 상태(결과 변수)**: 성과 발현의 양상과 결과 신호
- **내용 영역(독립 축)**: 개념 숙련도는 내용 영역 × 성과 상태로 해석

### 사전조사 5개 척도 최종 배치
| 척도 | 최종 위치 |
| --- | --- |
| 수학 불안 기본 반응성 | 마음 > 기질 > 정서 반응성 |
| 자기보고형 메타인지 인식 | 마음 > 신념 > 자기 개념(자기 인식 포함) |
| 학습 주도성 인식 | 마음 > 신념 체계 > 통제 가능성 신념 > 주도성 인식 |
| 실패 후 회복 기대 | 마음 > 신념 체계 > 회복 기대 신념 |
| 수학 정체성 인식 | 마음 > 신념 > 정체성 |

위 다섯 척도는 정신 상태가 아니라, 정신 상태를 해석하기 위한 마음 하위 기준값으로 둔다.

## 1. 통제 어려운 것 vs 통제 가능한 것 vs 성과 상태 — 최상위 분기의 정당성
이 분기는 단순한 편의가 아니라 **책임 소재를 분리하는 철학적 장치**다.  
통제 가능한 것과 통제 불가능한 것을 섞는 순간, 평가는 낙인이 되고 지도는 훈계가 된다.  
여기에 **성과 상태(결과 변수)**를 추가하면 프레임이 **조건–원인–결과** 구조로 닫힌다.

또한 **내용 영역(독립 축)**을 분리하면, 능력과 성과를 개념 맥락에서 안전하게 해석할 수 있다.

## 2. 통제 어려운 것(비개입 변수) 범주
서로 겹치지 않으면서도 전 영역을 포괄한다.

### A. 마음 (Temperament / Affective Disposition)
- 기질
- 성향
- 신념
  
**핵심 포인트**
- 조절은 가능하지만, 생성은 통제 불가
- “바꾸자”의 대상이 아니라 “이해하고 고려하자”의 대상

### B. 재능 (Aptitude / Cognitive Endowment)
- 처리 속도
- 작업기억 용량
- 공간 능력 등
  
**핵심 포인트**
- 상한선이 존재
- 단기적 변화 거의 없음
- 훈련 효과는 운용 방식에서 발생  
→ 조건 변수이며 평가 대상이 아님

### C. 운 (Chance / Contingency)
- 만난 문제 유형
- 시험 당일 컨디션
- 우연한 성공/실패 경험
  
**핵심 포인트**
- 평가에서 항상 노이즈
- 제거 불가, 통제 불가  
→ 반복 관찰이 필요

### D. 학습 환경 (Learning Environment)
- 교사
- 커리큘럼
- 가정/학교 맥락
  
**핵심 포인트**
- 학생 개인이 통제 불가
- 해석에는 반드시 반영

## 3. 통제 가능한 것 = 개입 가능한 것
의도적 훈련·설계·피드백으로 **변화시킬 수 있는 것**이다.  
따라서 평가의 주인공은 다음 3가지다.

- 통합적 수학 역량
- 정신
- 행동

## 4. 성과 상태 = 결과 변수(Outcome)
성과 상태는 학생이 직접 통제하지 못하지만, 교육적으로는 **개입의 핵심 지표**다.  
즉 **비개입 변수도, 개입 가능 변수도 아니며 결과 변수(Outcome)**로 분리해야 한다.

### 정의(추천)
성과 상태란, 특정 시점에서 통합적 수학 역량이 정신·행동·환경 조건 하에서
어떻게 발현되었는지를 보여주는 결과 지표이다.

### 핵심 효과
- 책임 소재가 완전히 분리된다(성과 낮음 ≠ 능력 낮음 ≠ 마음 문제).
- 성과를 판단이 아니라 **해석의 출발점**으로 만든다.
- 학생을 낙인찍지 않고, 개입 가능한 것만 개입한다.

## 5. 내용 영역(독립 축)
내용 영역은 성과 상태와 **직교 구조**를 이룬다.

### 핵심 공식
**개념 숙련도 = 내용 영역 × 성과 상태**

즉, 숙련도는 능력 자체가 아니라 **내용 맥락에서의 성과 발현**이다.

### 표현 예시
- ❌ “이 학생은 함수 능력이 약하다”
- ⭕ “이 학생은 함수 영역에서 사고·전략 발현의 안정성이 낮다”
- ⭕ “함수 개념에 대한 전이 성과가 제한적이다”

### 장점
- 개념 진단이 정확해진다(“무엇이 무너지는지”가 보임).
- 개입 포인트가 명확해진다.
- 확장성이 높다(내용 축·역량 축을 독립적으로 추가 가능).

## 6. “능력 지표 / 상태 지표” 이중 분해의 이유
- **작동 시간이 다름**: 상태는 단기 변동, 능력은 중·장기 축적
- **개입 방식이 다름**: 상태는 환경/정서 개입, 능력은 훈련/전략 개입
- **책임 분리가 가능**: 상태가 나쁨 ≠ 능력이 낮음

## 7. 통제 가능 영역 × (능력 / 상태) 매핑

### 1) 통합적 수학 역량
- **능력 지표**: 질문 구성 능력, 사고력, 논리력, 문제해결력, 메타인지
- **상태 지표**: 없음(원칙적으로)  
  역량은 정의상 안정적 능력에 해당한다.

### 2) 정신
- **능력 지표**: 정서 조절 능력, 동기 조절 능력, 인지적 회복력,
  과제 가치 인식(준안정 경향), 내재적 동기(준안정 경향), 지속 의지(준안정 경향)
- **상태 지표**: 불안 수준, 흥미, 즐거움, 좌절, 자신감, 긴장, 피로
  흥미는 성과 지표가 아니라 능력 발현을 조절하는 정서·동기 상태다.

### 3) 행동
- **능력 지표**: 전략 실행 능력, 오류 회복 능력, 지속성(문제 지속성)
- **상태 지표**: 시도 빈도, 회피 반응, 질문 빈도

## 8. 성과 상태 분해 구조(요약)
### 1차 분해(양상)
- 안정성, 범위, 전이성, 지속성

### 2차 분해(역량 발현)
- 사고 발현, 논리 발현, 문제해결 발현, 질문 발현, 메타인지 발현

### 보조 지표
- 점수, 성취도, 정답률
