# 학생 이해 프레임워크

이 문서는 **학생을 평가하기 위한 틀**이 아니라, **학생을 공정하게 이해하기 위한 틀**을 정리한다.

## 큰 구조(최상위 분기)
- **비개입 변수(통제 어려운 것)**: 마음, 재능, 운, 학습 환경
- **개입 가능 변수(통제 가능한 것)**: 통합적 수학 역량, 정신, 행동
- **성과 상태(결과 변수)**: 성과의 발현 양상과 결과 신호
- **내용 영역(독립 축)**: 개념 숙련도는 내용 영역 × 성과 상태로 해석

## 문서 구성
- `model.md`: 철학/분기 근거, 범주 정의, 능력·상태 분해
- `philosophy.md`: 교육철학 정리(요약/핵심 축/지향점)
- `criteria/`: 지표 정의(파일 1개 = 지표 1개)
- `terms/`: 용어 정의(파일 1개 = 용어 1개)
- `scoring.md`: 점수화/집계 규칙
- `scoring_snapshot_policy.md`: 출석/과제 스냅샷 운영 정책
- `scoring_migrations/`: 점수 규칙 변경 로그(마이그레이션 방식)
- `data_sources.md`: 데이터 소스 및 관찰 근거
- `glossary.md`: 용어 정리
- `decisions/`: 의사결정 로그
