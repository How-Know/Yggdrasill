# 좌절

## 정의
문제 해결이 막히거나 기대한 결과가 나오지 않을 때 생기는 **부정적 정서 상태**.
즉각적이며 상황 의존적인 상태 지표로 본다.

## 예시 / 반예시
### 예시
- 시도해도 풀리지 않아 감정이 급격히 떨어진다.
- 막히는 순간부터 참여가 급격히 줄어든다.

### 반예시
- 능력이 낮아서 항상 좌절한다고 단정한다.
- 일시적 피로나 외부 요인을 좌절로 일반화한다.

## 판단 기준
- 상황 변화에 따라 즉각적으로 상승/완화되는가?
- 회피/중단 같은 행동 반응과 연결되는가?
- 장기적 성향(기질)과 구분되는가?
