# 과제 가치 인식

## 정의
과제(문항, 단원, 학습 활동)를 **재미있고(흥미 가치), 유용하며(유용성 가치), 중요하다고** 인식하는 정도.
즉각적인 기분보다는 더 안정적이지만, 훈련으로 바로 바뀌는 능력과도 구분되는 **준안정 경향**에 가깝다.

## 예시 / 반예시
### 예시
- “이 단원은 나중에 꼭 쓰이니까 해볼 만하다.”
- “재미도 있고 의미도 있어서 해보고 싶다.”
- 성과가 조금 흔들려도 가치 인식이 유지된다.

### 반예시
- 그날 기분이 좋아서 잠깐 재미있다고 느낀 것을 가치로 오해한다.
- 단기 점수만 보고 과제의 의미를 판단한다.
- 외적 보상이 없으면 바로 가치 평가가 붕괴된다.

## 판단 기준
- 상황이 바뀌어도 어느 정도 유지되는가?
- 흥미(현재 상태)와 구분해 “왜 할 만한지”를 설명하는가?
- 설명/맥락 제공으로 점진적으로 변하는가?
