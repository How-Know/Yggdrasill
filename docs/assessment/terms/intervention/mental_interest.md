# 흥미

## 정의
수학 과제나 문제 해결 과정에서 활동 자체가 즐겁고 의미 있게 느껴지는 정서·동기 상태.
기분보다는 무겁고 능력보다는 가볍다. 성과의 조건을 설명하는 지표이지 성과 그 자체는 아니다.

## 예시 / 반예시
### 예시
- 새로운 문제를 보면 “풀어보고 싶다”는 느낌이 바로 생긴다.
- 어렵더라도 생각하는 과정이 즐거워 시도를 이어간다.
- 과제 외 시간에도 자발적으로 문제를 풀어본다.

### 반예시
- 성적이 높으니 흥미도 높다고 단정한다.
- 단순히 기분이 좋아서 그날만 반짝 흥미가 있는 경우를 일반화한다.
- 능력이 낮아서 흥미도 낮다고 결론 내린다.

## 판단 기준
- 상황 변화에 따라 비교적 즉시 변하지만, 하루 기분보다 더 일관성이 있는가?
- 성과와 독립적으로 관찰 가능한가? (흥미 ↑ ≠ 성과 ↑)
- 참여도(인지/정서/행동)의 상승으로 이어지는가?
- 과제 가치 인식·내재적 동기와 구분해 “현재 상태”를 묻고 있는가?
