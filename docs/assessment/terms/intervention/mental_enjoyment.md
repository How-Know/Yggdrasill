# 즐거움

## 정의
학습·문제 해결 과정에서 느껴지는 **즉각적이고 긍정적인 정서 상태**.
흥미와 밀접하지만, “현재의 기분/정서”에 더 가깝다.

## 예시 / 반예시
### 예시
- 문제를 풀면서 “재밌다”는 감정이 즉시 나타난다.
- 설명을 들을 때 긍정 정서가 올라가 집중이 잘 된다.

### 반예시
- 성적이 좋아서 기분이 좋은 것을 곧바로 즐거움 상태로 해석한다.
- 장기적 동기(내재적 동기)와 혼동한다.

## 판단 기준
- 상황 변화에 따라 빠르게 변하는가?
- 현재 정서 상태로 직접 관찰 가능한가?
- 흥미(동기)와 구분해 “감정의 톤”을 묻고 있는가?
