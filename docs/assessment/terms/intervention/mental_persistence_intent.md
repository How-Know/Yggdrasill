# 지속 의지

## 정의
단기 성과와 무관하게 **과제를 계속 이어가려는 의지와 태도**.
행동적 시도 빈도(상태)보다 느리게 변하며, 장기적 참여를 지탱하는 **준안정 경향**이다.

## 예시 / 반예시
### 예시
- 시간이 오래 걸려도 끝까지 풀어보겠다는 마음을 유지한다.
- 실패 경험이 있어도 다음 시간에 다시 시도한다.
- 계획을 조정해서라도 계속 이어가려 한다.

### 반예시
- 그날 기분이 좋아서만 잠깐 “열심히 하겠다”고 말한다.
- 외부 압박이 없으면 바로 포기한다.
- 시도 빈도만 높고, 장기적으로는 지속되지 않는다.

## 판단 기준
- 단기 성과 변화와 상관없이 지속 의지가 관찰되는가?
- “현재 시도”가 아니라 “앞으로도 이어갈지”에 대한 태도인가?
- 여러 세션에 걸쳐 비교적 일관된가?
