# 성과 상태 (Performance State)

## 본질
성과 상태는 학생이 직접 통제하지 못하지만, 교육적으로는 **개입의 핵심 지표**다.
시험 난이도, 문항 구성, 시간 압박, 평가 방식 등은 학생이 통제할 수 없다.
그러나 성과는 지도 전략 수정, 학습 방향 재설정, 피드백 설계의 출발점이 된다.

즉,
- 비개입 변수도 아니고
- 개입 가능 변수도 아니다  
→ **결과 변수(Outcome)**다.

## 최상위 분기 위치
- 비개입 변수(조건)
- 개입 가능 변수(원인)
- 성과 상태(결과)

이 구조를 두면 프레임이 **원인–조건–결과**로 닫힌다.

## 정의(추천)
성과 상태란, 특정 시점에서 통합적 수학 역량이 정신·행동·환경 조건 하에서
어떻게 발현되었는지를 보여주는 결과 지표이다.

## 명칭 후보
- Performance State
- Achievement State
- Observed Performance Snapshot

## 해석 원칙
- 성과가 낮다 ≠ 능력이 낮다 ≠ 마음이 문제다
- 성과 상태는 **판단의 근거가 아니라 해석의 출발점**

## 성과 상태 분해 원칙
1. 능력을 다시 정의하지 말 것  
   성과는 능력의 결과이지, 능력 그 자체가 아니다.
2. 원인과 섞지 말 것  
   불안·집중 같은 원인은 성과가 아니다.
3. “왜 이런 결과가 나왔는지”로 역추적 가능할 것  
   성과 분해는 해석을 위한 분해여야 한다.

## 분해 구조(요약)
### 1차 분해(양상)
- 안정성, 범위, 전이성, 지속성

### 2차 분해(역량 발현)
- 사고 발현, 논리 발현, 문제해결 발현, 질문 발현, 메타인지 발현

### 보조 지표
- 점수, 성취도, 정답률

## 한 문장 요약
성과 상태는 ‘얼마를 맞았는가’가 아니라,
‘어떤 능력이 어떻게 드러났는가’를 분석하는 구조여야 한다.
