# 상태 지표

## 정의
특정 학습·평가 상황에서 비교적 즉각적으로 관찰되며, 상황 변화에 따라 단기간에 변동하는 심리·인지 상태를 말한다.

## 핵심 속성
- 상황 의존적, 단기 변동성 큼
- 개입 없이도 변할 수 있음
- "얼마나/얼마나 자주"의 문제

## 판단 기준
- 상황이 바뀌면 즉시 바뀌는가? → 예면 상태 지표
- 의도적 전략 없이도 변하는가? → 예면 상태 지표
- 훈련 없이도 일시적으로 좋아질 수 있는가? → 예면 상태 지표
- 다른 과목/상황으로 전이된다면 능력 지표로 본다

## 관찰 원칙
- 주기적 관찰이 아니라, **변화가 감지될 때마다 기록**한다.

## 예시 / 반예시
### (1) 불안 관련 지표
- 수학 불안의 강도 — 상태 지표
- 수학 불안의 빈도 — 상태 지표
- 불안 인식 정확도 — 능력 지표
- 불안 속 사고 유지 — 능력 지표
- 불안 후 회복 속도 — 능력 지표

### (2) 집중/주의 관련 지표
- 현재 집중감 — 상태
- 피로도 — 상태
- 주의 유지 시간 — 능력
- 산만해질 때 복귀 전략 — 능력

### (3) 동기 관련 지표
- 오늘의 의욕 — 상태
- 과제 흥미 — 상태
- 목표 재설정 능력 — 능력
- 좌절 후 재도전 — 능력

## 주의사항
- 상태 ≠ 나쁨
- 상태 지표는 문제 진단용이며 낙인용이 아니다.
- 상태가 나빠도 능력이 높을 수 있다.
