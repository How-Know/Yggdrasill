# 회복 기대 신념

## 정의
실패해도 다시 잘할 수 있을 것이라고 믿는 기본 기대.
현재 의욕이나 회복 상태가 아니라, 회복이 가능하다는 **기본 프레임**이다.

## 예시 / 반예시
### 예시
- “수학에서 한 번 실패해도 다시 잘할 수 있다고 느낀다.”
- 실패 후에도 다시 시도할 수 있다는 기대가 유지된다.

### 반예시
- “한 번 막히면 계속 어려울 것 같다.”는 믿음이 지속된다.
- 오늘만 의욕이 올라온 것을 기대 신념으로 해석한다.

## 판단 기준
- 실패 이후에도 회복 가능성을 비교적 안정적으로 유지하는가?
- 현재 의욕/상태와 분리해 관찰 가능한가?
- 도전·회피 선택에 영향을 주는가?
