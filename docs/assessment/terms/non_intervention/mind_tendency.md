# 성향 (인지·행동 접근 경향)

## 정의
문제를 만났을 때 **어떤 방식으로 생각하고 행동하는 경향이 있는가**를 보여주는 접근 스타일이다.

## 구조
- 4개 독립 축: 자율성, 인지 처리, 실행 스타일, 도전 반응
- 2×2×2×2 = 16가지 성향 프로파일

## 추천 표현
- “프로파일링했다”
- “좌표화했다”
- “접근 경향을 구조화했다”

예:
“학생의 문제 해결 접근 성향을 4개 축에서 좌표화하여,
16가지 성향 프로파일로 표현한다.”

## 역할(시스템 내)
- 행동 패턴 예측자
- 성과 상태 해석 보정기
- 개입 전략 금지/권장선 표시기

## 무엇을 예측하는가
### 1) 행동 발현 패턴
- 회피적 + 직관형 → 질문은 머릿속에서만 하고 말로 잘 드러내지 않음
- 도전적 + 즉흥형 → 시도는 많으나 검증이 약할 수 있음

### 2) 성과 상태의 흔들리는 방향
- 계획적 + 논리형 → 안정성은 높으나 전이성은 낮을 수 있음
- 즉흥적 + 직관형 → 전이성은 높으나 안정성은 낮을 수 있음

### 3) 개입 전략의 금지선/권장선
- 자주적 + 도전형 → 과도한 통제는 역효과
- 의존적 + 회피형 → 초기 성공 경험 설계가 핵심

## 아니라고 명확히 해야 할 것
- 능력 평가 아님
- 성취 예측 점수 아님
- 지도 목표 아님

즉, **해석 보정용 ‘고정 좌표’**다.

## 연결 구조
[성향 조사 결과] → [정신·행동 상태 관찰] → [통합적 수학 역량 발현] → [성과 상태]

성향은 원인도 결과도 아니라 **경로를 휘게 만드는 힘**이다.

## 설명 문구(권장)
“이 조사는 아이의 실력이나 성취를 평가하는 것이 아니라,
문제를 만났을 때 어떤 방식으로 접근하는지를 이해하기 위한 조사입니다.
이후의 학습 결과를 해석하고,
아이에게 맞는 지도 방식을 선택하는 기준으로 사용됩니다.”

## 주의
MBTI처럼 보일 수 있으나 **성격 유형이 아니라 문제 해결 접근 스타일**이다.
단기간에 바꾸는 대상이 아니며, 등급화/평가용이 아니다.

## 한 문장 결론
“마음 하위에 문제 해결 접근 성향을 두고,
이를 네 개의 독립 축으로 구조화하여
16가지 성향 프로파일로 표현한다.”
