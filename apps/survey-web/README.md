# survey-web

Vite + React + TypeScript 기본 스캐폴드.

## 환경변수(Supabase)

설문 문항 로딩을 위해 아래 환경변수가 필요합니다.

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

로컬 실행 시:

- `env.local.example`을 `.env.local`로 복사한 뒤 `npm run dev`를 **재시작**하세요.
  - Vite는 실행 중 env를 다시 읽지 않습니다.

명령어
- 개발: `npm run dev`
- 빌드: `npm run build`
- 프리뷰: `npm run preview`
